Multi-tier architeture Project Deployment (EC2 + RDS + PHP)
# Main Project Highlights

Designed and implemented a Multi-Tier Architecture on AWS.
Deployed Frontend (Web Server), Backend (Application Server), and Database as separate layers.
Used Amazon EC2 instances for hosting web and application services.
Configured Amazon RDS MySQL for secure and managed database services.
Implemented Application Load Balancer (ALB) for efficient traffic distribution.
Created Amazon VPC with Public and Private Subnets for network isolation.
Configured Security Groups and IAM for secure access control.
Established secure communication between Web, Application, and Database layers.
Improved application availability and fault tolerance through distributed architecture.
Enabled database access only from the application layer for enhanced security.
Performed end-to-end testing to validate functionality and connectivity.
Applied cloud security best practices using IAM roles, private subnets, and controlled network access.
Achieved scalable, secure, and reliable application deployment on AWS.
Demonstrated practical implementation of cloud computing, networking, load balancing, and database management concepts.
Built a production-style architecture following industry-standard AWS deployment practices.
